# Accounting Workflows Documentation Library

Status: active scaffold
Last reviewed: 2026-05-21

This folder is a numbered, hierarchical documentation library.

- `00-*` foundations and navigation
- `10-*` operations and runbooks
- `20-*` automation and scheduling
- `30-*` consumer-facing guides
- `40-*` development/change guides
- `90-*` governance, ownership, and quality rules

Numbering convention:
- `00` to `09`: global index/start-here
- `10` to `19`: operations
- `20` to `29`: automation
- `30` to `39`: report consumption
- `40` to `49`: engineering/dev workflow
- `90` to `99`: governance/meta
